About

Technical documentation and system information.

How It Works

The system generates procedural mazes and analyzes user behavior during solving. Bot simulations test the detection accuracy by mimicking human-like and bot-like patterns.

Detection Methods

  • Solve time analysis
  • Path precision evaluation
  • Mouse movement patterns
  • Behavioral anomaly detection

Wall Tolerance

Maximum 3 wall touches per 10 path steps. Humans naturally make minor errors; bots typically follow perfect paths.
